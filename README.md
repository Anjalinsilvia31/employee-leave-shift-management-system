# Employee Leave & Shift Management System

## Overview
The Employee Leave & Shift Management System is a web-based application designed to help organizations manage employee leave requests and work shifts efficiently.

This system allows employees to:
- Apply for leave
- View leave status
- Check assigned shifts

Admins can:
- Approve or reject leave requests
- Manage employee shifts
- Monitor employee schedules through the dashboard

---

## Features

### Employee Features
- Apply for leave online
- View leave history
- Check shift schedules
- Dashboard for quick access

### Admin Features
- Approve/reject leave applications
- Assign and manage shifts
- Track employee attendance and schedules
